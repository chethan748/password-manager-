# Pass-Op Password Manager

A secure full-stack password manager built using React, Vite, Tailwind CSS, Node.js, and JWT authentication.

## Features
- User authentication
- Password encryption
- Password strength checker
- Secure credential storage
- Responsive UI

## Tech Stack
- React
- Vite
- Tailwind CSS
- Node.js
- JWT
- bcrypt

## Installation
npm install
npm run dev
